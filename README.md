# CalculatorErl
Программный калькулятор для оценки показателей качества обслуживания в моносервисной системе с явными потерями, то есть реализация модели Эрланга

Программа представляет собой оконное приложение с графическим интерфейсом, которое написано  на языке Python с использованием библиотек: thinker, math, csv, json, datetime.

Инструкция по применению
1.	Запуск приложения: двойным щелчком по файлу CalculatorErl.py (требуется установленные библиотеки thinker, math, csv, json, datetime и Python).
2.	Ввод данных: Выбор двух параметров, требуемых для задачи. Ввод значений согласно рекомендациям. Необходимо выбирать только два параметра, иначе вычисление заблокируется.
3.	Выполнение расчета: 
Нажмите кнопку «Вычислить». Алгоритм работы калькулятора:
 •	Проверка данных: программа проверяет корректность введённых значений (формат, диапазоны, логические ограничения).
 •	При ошибке: если данные некорректны, выводится сообщение об ошибке красным цветом, расчёт не производится.
 •	При успехе: выполняется расчёт неизвестных параметров.
Результаты расчёта отображаются следующим образом:
 •	В полях ввода: все четыре поля (включая ранее заблокированные) заполняются вычисленными значениями.
 •	В таблице истории: новая строка добавляется в таблицу с указанием: № операции, v, a, p, m, время расчёта.
4. Очистка:
 •	Кнопка «Очистить поля» Сбрасывает все текстовые поля в нулевое значение, снимает все флажки выбора параметров. Калькулятор возвращается в исходное состояние..
 •	Кнопка «Очистить историю» Удаляет все записи из таблицы истории вычислений. Требует подтверждения действия. Не влияет на поля ввода и текущие результаты.
5. Экспорт данных
Для сохранения истории вычислений:
 •	Нажмите кнопку «Экспорт CSV».
 •	В диалоговом окне выберите папку и укажите имя файла.
 •	Файл будет сохранён в формате CSV с кодировкой UTF-8.
 •	Файл можно открыть в Microsoft Excel, LibreOffice Calc или любом текстовом редакторе.
6.	Завершение работы: закрыть окно программы стандартным способом.
Закройте окно программы стандартным способом:
 •	Нажмите на крестик в правом верхнем углу окна, или
 •	Нажмите клавишу Esc (если активные поля ввода не мешают обработке этой клавиши).
